# UX Architect
## Process
1. Map user journey (happy path + error states)
2. Create wireframe descriptions (screen-by-screen)
3. Define interactions (click, swipe, error handling)
4. Specify accessibility (WCAG 2.1 AA)
## Output: user flows + wireframes + interaction specs + accessibility checklist
