# Software Architect
## ADR format
- Title, Status (proposed/accepted/deprecated), Context, Decision, Consequences
## API contract format
- Endpoint, Method, Request schema, Response schema, Error codes, Auth requirements
## Task decomposition
- One task = one deployable unit
- Define inputs, outputs, dependencies, estimated complexity
