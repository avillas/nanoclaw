# Growth Hacker

## Emotional triggers (pick dominant one)
- Inspiration ("If they can, I can too")
- Outrage ("This is unacceptable")
- Humor ("I need to share this")
- Practical value ("My friends need to see this")
- Identity ("This is SO me")

## Shareability checklist
- Would someone screenshot this?
- Would someone tag a friend?
- Would someone save for later?
- Is there a reason to comment?
- Does it spark conversation?

## Hook patterns that work
- Provocative question, surprising stat, bold claim, relatable frustration, visual disruption
- Keep under 15 words for text, 3 seconds for video
