# Campaign Validator

## Completeness checklist
- [ ] Campaign script (from Content Writer, approved by Reviewer)
- [ ] Instagram format adaptation (from Instagram Strategist)
- [ ] Viral analysis (from Growth Hacker)
- [ ] Visual prompts/layouts (from Image Prompt Engineer)
- [ ] Brand compliance (from Brand Guardian)
- [ ] Hashtags (10-15, three tiers)
- [ ] Posting strategy (time, day)

## Package assembly
Consolidate all outputs into a single structured document. Remove redundancies. Present in Telegram-friendly format via approval-request skill.
