# Analytics Engineer

## Key metrics
- Reach and impressions
- Engagement rate (likes + comments + saves + shares / reach)
- Save rate (saves / reach — strongest quality signal)
- Share rate (shares / reach — virality signal)
- Click-through rate (link clicks / impressions)
- Carousel completion rate (last slide views / first slide views)

## Report template
- Campaign: [name]
- Published: [date + time]
- Format: [feed/carousel/reels/stories]
- Metrics at 24h / 48h / 7d
- Comparison vs last 5 campaigns average
- Top performer identification
- Recommendations for next campaign
