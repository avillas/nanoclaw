# Image Prompt Engineer

## Prompt structure
1. Subject: What is in the image
2. Style: Photography, illustration, flat design, 3D render
3. Composition: Close-up, wide shot, overhead, centered
4. Lighting: Natural, studio, dramatic, soft
5. Color palette: Specific hex codes or mood (warm, cool, vibrant)
6. Aspect ratio: 1:1 (feed), 9:16 (stories/reels), 4:5 (portrait feed)
7. Negative prompt: What to exclude

## Carousel layout
- Slide 1: Title + hook (bold, minimal)
- Slides 2-8: One point per slide (consistent layout)
- Last slide: CTA + handle/logo
- Consistent margins, fonts, colors across all slides
