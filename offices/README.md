# NanoClaw Multi-Office — Wave 1

## Installed files

- 3 office CLAUDE.md (marketing, development, innovation)
- 3 entry-point agents (content-writer, product-manager, trend-researcher)
- 3 entry-point skills (content-creator, product-manager, trend-researcher)
- 10 shared skills
- 3 templates

## Directory structure

```
offices/
├── marketing/     (CLAUDE.md + 1 agent + 1 skill)
├── development/   (CLAUDE.md + 1 agent + 1 skill)
├── innovation/    (CLAUDE.md + 1 agent + 1 skill)
├── shared/skills/ (10 shared skills)
└── _template/     (3 templates for new offices)
```
