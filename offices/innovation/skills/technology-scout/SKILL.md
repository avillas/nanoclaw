# Technology Scout
## Evaluation criteria
- Maturity: production-ready? beta? experimental?
- Community: GitHub stars, contributors, Stack Overflow activity
- Documentation: quality, completeness, examples
- Maintenance: last commit, release frequency, issue response time
- Fit: compatible with current stack? learning curve?
## Technology radar categories: Adopt | Trial | Assess | Hold
