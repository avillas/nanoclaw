# Daily Report

## Format

Completed: [count + list]
In progress: [count + list]
Blocked: [count + reasons]
Cost: Today R$ X / R$ Y (Z%) | Month R$ X / R$ Y (Z%)
Model mix: Haiku X% | Sonnet X% | Opus X%
