# Pull Request

## Template
- What: [1-2 sentence summary]
- Why: [task/story reference]
- Changes: [list of files and what changed]
- How to test: [step-by-step]
- Checklist: conventions, tests, no secrets, no debug statements, docs updated
## Rules
- Target: develop (never main)
- One PR per task
- Human approval required for merge
