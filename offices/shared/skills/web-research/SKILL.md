# Web Research

## Process
1. Define the question
2. Plan 3-5 search queries before executing any
3. Execute searches (max per office rate limit)
4. Extract: URL, key data point, relevance
5. Synthesize into coherent summary
6. Cite every claim

## Rules
- Prefer recent sources (last 6 months)
- Prefer primary sources over aggregators
- Every finding must cite its source
