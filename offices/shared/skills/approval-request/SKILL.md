# Approval Request

## Format
📋 Approval request: [Office]
[Title]
Summary: [3-5 bullet points]
Options: ✅ Approve | ✏️ Request changes | ❌ Reject

## Rules
- Keep under 500 characters
- Include ONLY decision-relevant information
- Always state what happens on approval
